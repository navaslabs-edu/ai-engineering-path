# 01 - FastAPI Fundamentals

## Objective

Get completely comfortable building an API with FastAPI - routes, path/query parameters, request bodies, response models, and error handling - before any AI model enters the picture.

## Prerequisite

[00 - Prerequisites](../00-prerequisites/)

## What you'll build

A small **Campus Helpdesk API** that answers student questions with fixed, hand-written answers. It looks and behaves like a real product endpoint - the only thing missing is real intelligence behind it.

## Files in this folder

- `notes/Session1_FastAPI_Fundamentals.md` - the full written lesson
- `starter/main.py` - skeleton to build from
- `solution/main.py` - the reference build
- `exercises.md` - practice questions (Parts A–E)

## How this connects forward

In [02 - Calling LLM APIs](../02-calling-llm-apis/), we take this exact file and replace the fixed answer inside `ask_question()` with a real call to an AI model. Nothing else about the API changes. Keep your solution from this session - you'll need it.
