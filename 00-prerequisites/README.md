# 00 — Prerequisites

## Objective

Confirm you're ready to start the AI Engineering Path. This is a hard gate, not a soft suggestion — sessions from 01 onward assume you already have these skills, and the course moves fast because of it.

## What you need before joining

- **Python**: variables, functions, classes (basic OOP), file handling, virtual environments
- **Git & GitHub**: clone, commit, branch, push, pull request
- **SQL**: SELECT/INSERT/UPDATE/DELETE, basic JOINs
- **Comfort with the terminal/CLI**

## How to check yourself

If you can build a small CLI tool that reads a CSV, stores rows in a SQLite database, and you can push that code to GitHub via a branch and pull request without looking up basic Git commands — you're ready for Session 01.

## If you're not ready yet

That's completely fine — better to spend two to three weeks here than to struggle through every session after. Recommended order:

1. Python fundamentals (functions through file handling)
2. Git & GitHub basics
3. SQL fundamentals

Ask in the NavasLabs community for the specific beginner tracks if you need structured material for any of these — they exist as separate NavasLabs repos.
